K.L.'s Tanx

Alpha 2010.7.24

 

 

l  Work Platforms: Windows XP, Windows Vista, Windows 7.

l  Plugin for web browser: Google Chrome, Firefox, Internet Explorer.

  

 

Beside the web browser plug-in, you can also:

 

l  Tanx Launcher: the main entry, to browse, preview and launch Tanx games offline.

l  Tanx Screen Saver: a screen saver program based on contents of Tanx games.

 

And tools:

l  TanxEditor.exe: a generic editor, see the illustration: TanxEditorQuickStart.png.

l  TanxGamePacker.exe: to pack game resource files.
