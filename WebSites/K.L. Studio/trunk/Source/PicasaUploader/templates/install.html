Install K.L.'s Picasa Uploader

  1. Dowload and install Picasa.

  2. Add the K.L. Uploader to Picasa.

    Tips: if the button can't be added, try this:

    1. Close Picasa.
    2. Execute the dos command:
      move %localappdata%\Google\Picasa2\buttons\KLPicasaUploader.pbz "[Picasa Directory]\buttons\"
      [Picasa Directory] is your picasa installed program direcotry.
    3. Reopen Picasa and retry add the "K.L. Studio" button.
