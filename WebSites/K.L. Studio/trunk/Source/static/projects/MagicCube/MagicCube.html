Script Programmability

You can also play the magic cube by writing scripts (lua or python). This is a sample in lua:

g_Cube = nil
g_Steps = 0
g_Route = {L2, R2, U2, D2, F2, B2}

function initialize(cube)
	g_Cube = cube
end

function step()
	g_Steps = g_Steps + 1
	if g_Route[g_Steps] == nil then
		return 0
	end

	return g_Route[g_Steps]
end
More about scripting, look at `Scripts Guide.txt' in game directory.
