K.L.'s Tetra

 

Available for Windows XP/Vista/7

 

Click here to browse Source Code

Tetra is the 3D extension of the board game ‘Chinese Checkers’. The wire frame’s shape is the extended version of hexangular star in 3D space. It’s the combination of 2 intersected tetrahedrons, seems like a diamond.

K.L.'s Tetra is an open-source project, see the home page on SourceForge.net.

Thanks to these third-party softwares: Boost, OGRE, CEGUI, Lua, Luabind, wxWidgets
